## Tag Reference

Apply tags from each relevant category. Multiple tags per category are fine.

### Meal Type
| Tag | Use for |
|-----|---------|
| `#breakfast` | Morning meals |
| `#lunch` | Midday meals |
| `#dinner` | Evening meals |
| `#snack` | Between-meal bites |
| `#dessert` | Sweet finishes |

### Prep Style
| Tag | Use for |
|-----|---------|
| `#mealprep` | Designed to be made in bulk and stored |
| `#quickmeal` | Under 30 minutes total |
| `#slowcooker` | Crock pot / slow cooker required |
| `#airfryer` | Air fryer required |
| `#grill` | Grilled or smoked |
| `#bake` | Oven-baked |
| `#onepan` | Single pan/pot cleanup |

### Protein Source
| Tag | Use for |
|-----|---------|
| `#chicken` | Chicken breast, thighs, ground |
| `#beef` | Ground beef, steak, brisket |
| `#pork` | Pork loin, bacon, ham |
| `#fish` | Salmon, tilapia, cod, etc. |
| `#shrimp` | Shrimp / shellfish |
| `#eggs` | Egg-based dishes |
| `#vegetarian` | No meat |

### Macro Profile
| Tag | Use for |
|-----|---------|
| `#highprotein` | 40g+ protein per serving |
| `#lowcarb` | Under 20g carbs per serving |
| `#highcarb` | 60g+ carbs per serving |
| `#bulking` | High calorie, designed for a surplus |
| `#cutting` | Lower calorie, higher protein-to-calorie ratio |
| `#balancedmacros` | Roughly equal protein/carb/fat ratio |

### Format
| Tag | Use for |
|-----|---------|
| `#pasta` | Pasta-based dishes |
| `#soup` | Soups, stews, chili |
| `#salad` | Cold salad-style dishes |
| `#sandwich` | Sandwiches, wraps, tacos |
| `#bowl` | Rice/grain bowls |
| `#casserole` | Baked casserole or mac-style dishes |
| `#smoothie` | Blended drinks |

### Effort Level
| Tag | Use for |
|-----|---------|
| `#easy` | Minimal steps, hard to mess up |
| `#moderate` | Some technique or timing required |
| `#advanced` | Multiple components, skilled technique |
